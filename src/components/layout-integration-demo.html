布局集成演示

展示 RightSidebar 组件在实际页面中的布局效果

布局结构说明

✅ 桌面端（≥768px）：主内容区右侧留出 260px 空间，边栏固定在右侧

✅ 移动端（<768px）：主内容区占满宽度，边栏自动隐藏，显示底部导航

✅ Flex 布局：主内容区自动适应剩余空间

✅ 平滑过渡：窗口大小改变时有 300ms 动画

app/layout.tsx 代码 

import RightSidebar from '@/src/components/RightSidebar';
import { getCurrentUser } from '@/lib/auth';

export default async function RootLayout({ children }) {
  const user = await getCurrentUser();
  
  const env = {
    SUPABASE_URL: process.env.NEXT_PUBLIC_SUPABASE_URL || '',
    SUPABASE_ANON_KEY: process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY || '',
  };

  return (
    <html>
      <body>
        <div className="flex">
          {/* 主内容区 - 右侧留出边栏空间 */}
          <main className="flex-1 md:mr-[260px]">
            {children}
          </main>
          
          {/* 右侧边栏 */}
          <RightSidebar env={env} user={user} />
        </div>
      </body>
    </html>
  );
}

关键 CSS 类说明

flex-1

Flex 布局：占据剩余空间

md:mr-[260px]

桌面端：右边距 260px（与边栏宽度一致）

transition-all duration-300

平滑过渡：300ms 动画效果
