RightSidebar 组件演示

展示固定右侧边栏的功能，包括搜索、分类导航、授权用户入口等

主要特性

  • ✓
    固定定位： 右侧边栏固定在屏幕右侧，宽度 260px
  • ✓
    透明模糊： 半透明背景 + backdrop-blur 模糊效果，现代感十足
  • ✓
    搜索功能： 顶部搜索栏，支持实时输入和提交搜索
  • ✓
    官方分类： 从 Supabase categories 表读取，自动显示分类链接
  • ✓
    授权入口： 当用户 is_authorized=true 时，显示"我的管理主页"等管理入口
  • ✓
    响应式设计： 桌面端显示右侧栏，移动端改为底部导航栏

使用场景

内容网站

博客、新闻站、论坛等，提供分类导航和搜索功能

社区平台

用户管理、权限控制、快速入口等功能

电商应用

商品分类、用户中心、购物车快捷入口

管理系统

侧边栏导航、用户权限管理、系统设置

响应式设计

桌面端 (≥768px)

右侧固定边栏，260px 宽度，完整显示所有功能

移动端 (<768px)

隐藏右侧边栏，显示底部导航栏，分类通过模态框显示

💡 提示：调整浏览器窗口大小或使用开发者工具的响应式模式，查看不同屏幕尺寸下的效果

基础用法 

import RightSidebar from '@/src/components/RightSidebar';

// 准备环境变量
const env = {
  SUPABASE_URL: process.env.NEXT_PUBLIC_SUPABASE_URL!,
  SUPABASE_ANON_KEY: process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!,
};

// 获取当前用户（可选）
const user = await getCurrentUser();

// 使用组件
<RightSidebar env={env} user={user} />
