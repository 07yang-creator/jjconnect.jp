```bash

# 部署
vercel --prod

# 配置环境变量
vercel env add NEXT_PUBLIC_SUPABASE_URL
vercel env add NEXT_PUBLIC_SUPABASE_ANON_KEY
```

#### 选项2: Cloudflare Pages（HTML版本）
```bash
# 通过Wrangler部署
wrangler pages publish ./ --project-name=jjconnect

# 或在Cloudflare Dashboard上传文件
```
